## Run Locally

Clone the project

```bash
  git clone https://github.com/dask-58/dask-58.github.io.git
```

Go to the project directory

```bash
  cd dask-58.github.io
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
